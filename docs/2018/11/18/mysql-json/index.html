查询

表达式： 对象为json列->'$.键', 数组为json列->'$.键[index]'

等价于：对象为JSON_EXTRACT(json列 , '$.键')，数组为JSON_EXTRACT(json列 , '$.键[index]')

不过看到上面"小萌"是带双引号的，这不是我们想要的，可以用JSON_UNQUOTE函数将双引号去掉
