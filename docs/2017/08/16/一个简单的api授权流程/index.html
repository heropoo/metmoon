一个简单的API授权流程

2017-08-16

字数统计: 909   |   阅读时长≈ 4 分钟

实际项目中我们经常会碰到给第三方开放我们项目API的这种场景。这时我们要保证api的安全，参考腾讯广点通的api调用。现将php的代码实现整理这此。

我们的算法很简单：

  1. 把调用方自己的appid,secret_key,当前的时间戳time连接起来用sha1方法生成一个sign
  2. 把appid,time,sign用英文逗号连接并用base64打包变成一个参数token
