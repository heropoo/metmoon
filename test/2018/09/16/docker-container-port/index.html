方法1 修改iptables端口映射

docker的端口映射并不是在docker技术中实现的，而是通过宿主机的iptables来实现。通过控制网桥来做端口映射，类似路由器中设置路由端口映射。

比如我们有一个容器的80端口映射到主机的8080端口，先查看iptables到底设置了什么规则：
